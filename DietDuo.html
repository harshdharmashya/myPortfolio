DietDuo

DietDuo is a healthy meal planner website that helps users plan their meals and workouts, offering personalized meal suggestions based on meal type and weekday.

Home>Info>DietDuo

Features

User-Friendly Interface

DietDuo is easy to use application and also user store their data on their profile.

authentication & authorization

Implemented secure user registration and login functionalities build using firebase authentication system.

Meal Planning

Users can add meals categorized by meal type (breakfast, lunch, and dinner) to specific weekdays and also meals are update on every refresh.

Workout Scheduling

Users can schedule workouts according to difficulty levels (beginner, intermediate, and expert) for each weekday.

Detailed Information

A modal feature is available to display more details about meals and exercises.

Profile

After login User can see their information in profile section here i'm using firebase Cloud database.

Notifications

Incorporated notifications on clicks to provide users with real-time feedback and updates on their actions.

About

This section offers information about DietDuo, including its services, features, and contact details.

Navbar

The navbar connects to the Meal, Workout, About, and Home pages by using React Feature Router
