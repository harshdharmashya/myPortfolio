ezBook

ezBook is a user-friendly book store website that makes it easy for anyone to browse and buy books online. It features a clean design, simple navigation, and a wide range of book categories. Users can quickly find their favorite books. This project showcases essential frontend skills and a smooth, enjoyable shopping experience.

Features

User-Friendly Interface

ezBook is a book ordering UI web application that has been built using React,React Toolkit and Bootstrap.

authentication & authorization

Implemented secure user registration and login functionalities which allow user to add books to cart

Add to Cart

Implemented features such as "Add to Cart" functionality to allow users to easily add books to their shopping cart.

Notifications

Incorporated notifications on clicks to provide users with real-time feedback and updates on their actions.

About

Created an informative "About" section to provide users with background information on the website and its purpose.

View Details

Implemented a "View Details" section to allow users to access in-depth information about each book for better decision making.

Dark Mode

Provides a user-friendly interface with the option to switch to dark mode for a more comfortable viewing experience in low-light environments.

Active Contact Form

Have a question or need assistance? Our active contact form allows you to easily drop a message to the owner, ensuring you get timely responses and support.
