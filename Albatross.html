Albatross-Travel

An immersive travel website that offers comprehensive travel planning, destination guides, and easy booking for your next adventure.

Features

User-Friendly Interface

Albatross-Travel is a Travel Blog UI web application that has been built using Python, Django and Bootstrap.

authentication & authorization

Implemented secure user registration and login functionalities using Django's built-in authentication system.

Forgot Password

Included a "Forgot Password" feature that allows users to reset their password via email verification, ensuring secure and convenient account recovery.

Pagination

Implemented pagination to efficiently navigate through blog posts.

Read More

Developed a "Read More" feature for blog posts that displays a brief excerpt with an option to expand and read the full article, providing a clean and concise user interface.

Responsive

Ensured the website is fully responsive using Bootstrap and Media Query.

Search

Integrated search and filtering capabilities to allow users to quickly find specific blog posts.
